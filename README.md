### Smile Tech

### 背景（製品開発のきっかけ、課題等）
小学生の子どもを持つ親は、子どもが学校で元気に過ごせているのか不安です。
学校は楽しいだろうか？登下校は安全だろうか？などなど、目の届かないところにいる子どもはとにかく心配でたまりません。

最近ではスマートフォンを持ってきてもよいとする小学校も増え、小学生もスマートフォンを持ち歩く時代になりました。
スマートフォンの機能を駆使すれば、子どもの動向を常に確認することも可能です。しかし、それでは子どもは「監視」されているようであまりいい気持ちはしないでしょう。

どうすれば子どものプライバシーを守りつつ、親に安心を届けられるだろうか？その答えとして考えられたのが「エミル」です。

### 製品説明（具体的な製品の説明）
「エミル」は子どもが学校に行っている間の笑顔の状況を確認できるアプリです。子どもが笑うとデバイスがそれを検知し、親は子どもが学校で元気に過ごしていることを簡単に確認できます。
