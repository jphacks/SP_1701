## 製品概要
### Smile Tech
エミルは、子どもが学校に行っている間の笑顔の状況を確認できるアプリです。子どもが笑うとデバイスがそれを検知し、親は子どもが学校で元気に過ごしていることを簡単に確認できます。

### 背景（製品開発のきっかけ、課題等）
小学生の息子・娘がいる家庭の親は、子どもが学校で元気に過ごせているのかという不安があるのではないかと私たちは考えました。
また最近は、届出を出せば子どもにスマートフォンを持たせても良いという体制をとる小学校もあるそうです。そこで、子どものスマートフォンをHackして何かできないかと考えました。
